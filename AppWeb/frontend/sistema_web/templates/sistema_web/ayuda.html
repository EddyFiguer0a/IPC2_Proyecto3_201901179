Manual de Usuario
1. Cargar Configuración

Para iniciar, debes cargar un archivo XML con la configuración del sistema que incluya recursos, categorías, clientes e instancias.

  1. Ve a la sección "Cargar XML"
  2. Selecciona el archivo de configuración
  3. Haz clic en "Cargar Configuración"
2. Cargar Consumos

Una vez cargada la configuración, puedes registrar los consumos de recursos de cada instancia.

  1. Ve a la sección "Cargar XML"
  2. Selecciona el archivo de consumos
  3. Haz clic en "Cargar Consumos"
3. Generar Facturas

El sistema generará automáticamente facturas por cada cliente según los consumos registrados.

  1. Ve a la sección "Generar Facturas"
  2. Ingresa el periodo (fecha inicio y fecha fin)
  3. Haz clic en "Generar Facturas"
  4. El sistema calculará los costos y creará las facturas
4. Ver Datos

Puedes consultar toda la información del sistema en cualquier momento.

  • Recursos disponibles
  • Categorías registradas
  • Clientes activos
  • Instancias creadas
  • Consumos registrados
  • Facturas generadas
5. Generar Reportes PDF

Crea reportes profesionales en formato PDF.

  • Detalle de Factura: Genera un PDF con el desglose completo de una factura
  • Análisis de Ventas: Genera un reporte con las categorías y recursos que más ingresos generan
6. Reiniciar Sistema

Si deseas eliminar todos los datos y comenzar desde cero, usa la opción "Reiniciar Sistema" en el menú superior.

Documentación Técnica
Arquitectura del Sistema

El sistema está dividido en dos componentes principales:

Backend (Flask)
  • API REST para procesar solicitudes
  • Almacenamiento de datos en archivos XML
  • Validación con expresiones regulares
  • Programación Orientada a Objetos (POO)
Frontend (Django)
  • Interfaz web para el usuario
  • Comunicación con la API mediante HTTP
  • Generación de reportes PDF
  • Validación de formularios
Tecnologías Utilizadas
  • Python 3.x: Lenguaje de programación
  • Flask: Framework para el backend
  • Django: Framework para el frontend
  • Bootstrap 5: Framework CSS
  • xmltodict: Procesamiento de XML
  • ReportLab: Generación de PDF
